The 'tg' folder has the following structure:

tg-source	- Files directly related to outside TG test data.
 imported	- Contains the original test group data from standard body.
 		  TG01-TG27 are not supplied and you have to get them
		  from the standard body.
                  
 generated      - Contains .978 generated from 'imported' files. Also
 		  has additional test groups generated by me.

results         - Files created during testing

triggers	- Files containing start dates for testing files and times
 		  when specific testing steps are to be taken.
